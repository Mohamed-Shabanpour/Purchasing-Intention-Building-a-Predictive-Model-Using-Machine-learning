# Purchasing-Intention-Building-a-Predictive-Model-Using-Machine-learning

This repository contains a project report on predictive modeling of customer purchasing intention using machine-learning techniques.

The project was conducted as an applied analytics study and focuses on:
- feature engineering and selection,
- handling class imbalance,
- model evaluation and comparison.

The report documents the full methodology, results, and managerial insights.
The data used in this project was developed for industrial research purposes and is not included here in a standalone reproducible form.
